## Números primos.

```
Inicio numerosPrimos
n = 0
Mientras n <=100
    si n > 1
        si (n mod 2) = 0 
            escribir n + " no es primo."
        sino
            si (n mod n) = 0
                escribir n + " es primo."
            finsi
        finsi
    sino
        si n > 0
            escribir "1 es primo."
        finsi
    finsi
    n = n + 1
Fin Mientras
Fin numerosPrimos

```
Inicio calculadora

res = 0
sum1 = 0
sum2 = 0
Escribir "Elegir: 1. Sumar, 2. Restar, 3. Multiplicar o 4. Dividir"

si num = 1 o num = 2 o num = 3 o num = 4

            Escribir "Inserte los dos números a sumar"
